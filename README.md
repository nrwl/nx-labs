# Nx Labs
(test)

This repo contains experimental Nx plugins that are maintained by the Nx core team.

## Adding a plugin

```bash
nx g @nx/plugin:plugin [new plugin]
```

## Publishing a plugin

```bash
nx publish [plugin] -ver=[version] --tag=[tag]


# For example
nx publish nx-remix --ver=1.0.0-alpha.2 --tag=next
```
